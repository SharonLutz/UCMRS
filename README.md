# UCMRS
Unmeasured confounding in the MR Steiger approach

## Installation
```
install.packages("devtools") # devtools must be installed first
install.packages("psych") # if not already installed 

devtools::install_github("SharonLutz/UCMRS")
```
## Example
```
library(UCMRS)
?UCMRS
UCMRS(nSim=50)
```
